# 🎬 PhimHub - Movie Streaming & Management Platform

> A dynamic and secure web application for streaming movies, built from scratch using Java Servlet/JSP and Hibernate. 

## 📝 About The Project

PhimHub is a full-stack Java web application designed to provide users with a seamless movie-watching experience while offering administrators a powerful dashboard for content management. This project was developed to deeply understand the core concepts of Java EE (Servlets, Filters, Sessions) and Object-Relational Mapping (ORM) before transitioning to modern frameworks like Spring Boot.

## ✨ Key Features

### 👤 User Features:
* **Authentication:** Secure Registration, Login, and Password Recovery via **Email OTP** (JavaMail API).
* **Movie Experience:** Watch movies, add to Favorites, and share movies with friends via Email.
* **Smart Assistant:** Integrated **Gemini AI API** to provide an intelligent chatbot that suggests movies and answers user queries.
* **Payment Integration:** Basic checkout and payment flow for premium content.

### 🛡️ Admin Features:
* **Video Management:** Full CRUD operations for movies (Add, Edit, Delete, Update status).
* **User Management:** View and manage user accounts.
* **Statistical Dashboard:** Generate reports on video views, favorite counts, and user engagement.
* **Security:** Role-based access control implemented via **Java Filters** to prevent unauthorized access to admin routes.

## 🛠️ Built With

* **Backend:** Java EE (Servlet, JSP, JSTL), Hibernate / JPA
* **Database:** MySQL
* **Frontend:** HTML, CSS, JavaScript, Bootstrap
* **Server:** Apache Tomcat
* **Build Tool & Dependency:** Maven
* **Third-Party APIs:** Google Gemini AI API, JavaMail API
